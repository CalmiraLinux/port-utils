Утилита для работы с системой портов: использование

port-utils - скрипт для работы с системой портов Calmira Linux. Он умеет обновлять и устанавливать порты, просматривать список изменений в портах, а так же обновлять документацию дистрибутива Calmira Linux

Главной особенностью скрипта является его простота и небольшой вес. Он сможет работать на слабом оборудовании ввиду своей простоты. На данный момент, утилита совместима только с Calmira LX4 1.1.

Зависимости

Список зависимостей у port-utils небольшой:

  • requests - модуль Python для работы с сетевыми запросами. Установка: 

    su -c "pip3 install requests"

  • Tkinter - уже встроен в Python. Сейчас используется крайне редко для отображения графических окон с информацией о программе, не более.

Как вы могли заметить, скрипт довольно минималистичный. И на сегодняшний день он выполняет всего несколько обязанностей:

  • Скачивание портов и их установка (ветки testing и stable)
  • Просмотр новостей - списка изменений (добавление или удаление портов, изменение в составе и пр.)
  • Обновление документации дистрибутива

Опции

  • --tree stable/testing/list - нужная ветка, с которой скачивать обновления портов, либо же просмотр списка веток;
  • --doc stable/testing - нужная ветка, с которой скачивать документацию;
  • --news stable/testing - ветка, с которой нужно просмотреть список изменений в портах;
  • --about - просмотр информации об утилите в графическом и консольном режимах соотв.
